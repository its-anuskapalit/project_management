What’s inside (high-level)
Agile Project Management

Agile principles, Scrum roles & ceremonies, Kanban basics, and practical tips for applying Agile in projects.

Foundations and Initiation

Project definitions, project vs. program vs. portfolio, stakeholders, objectives, project charter, and initiation checklists.

Project Schedule

Work Breakdown Structure (WBS), Gantt charts, Critical Path Method (CPM), PERT analysis, scheduling best practices and examples.

Course PDFs

CSEB422.pdf — course/resource material for CSEB422.

Project Management - LPU - CSEB442.pdf — LPU course notes.

UNIT I-III.pdf and UNIT IV-VI.pdf — consolidated unit-wise notes for quick revision.

How to use this repo

Browse folders in GitHub or clone the repo:

git clone https://github.com/its-anuskapalit/project_management.git
cd project_management


Open the PDFs directly in GitHub or download them for offline study.

Use the Project Schedule folder to find scheduling worksheets and examples you can adapt for assignments.

Suggested study flow

Start with Foundations and Initiation to understand fundamentals and terminology.

Read UNIT I-III.pdf for the early units and theory.

Move to Project Schedule for planning and scheduling techniques.

Study Agile Project Management to contrast traditional and Agile practices.

Finish with UNIT IV-VI.pdf and CSEB422.pdf/CSEB442.pdf for advanced topics, quality, risk, procurement, and modern frameworks.
